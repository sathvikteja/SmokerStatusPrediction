# Smoker Status Prediction

This project predicts whether a person is a smoker based on health-related features
using machine learning models.

## Files
- SmokerStatusPrediction.ipynb – Main notebook
- SStest.ipynb – Testing notebook
- SmokerStatusPrediction.pdf – Project report

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn
